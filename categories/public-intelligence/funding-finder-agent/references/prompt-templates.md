# Prompt Templates

## General discovery

```text
Use $funding-finder-agent to find EU grants and funding programs for:
- country: Germany
- stage: seed
- sector: AI
- trl: 5
Return bilingual output with eligibility fit and deadlines.
```

## Deadline-focused

```text
Use $funding-finder-agent to list EU innovation funding opportunities for a climate-tech startup in Spain, and prioritize deadlines in the next 90 days.
```

## No-match diagnostic

```text
Use $funding-finder-agent to evaluate if my govtech startup profile fits current EU grants. If no strong match exists, explain blockers and what to change.
```
